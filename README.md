# Restaurant Menu Display

A React app showcasing a restaurant menu with reusable components and list rendering, built with Vite.

## Components
- `MenuItem.jsx`: Reusable component to display individual menu items with conditional vegetarian badge.

## Links
- **GitHub Repository:** [https://github.com/shahansheikh/restaurantmenu_shahansheikk.git]
- **Live Website:** [https://restaurantmenu-shahansheikk-sfne.vercel.app/]

## Run Locally
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
